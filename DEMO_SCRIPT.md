# Demo Script — AI Placement Intelligence Platform

5-minute walkthrough for interviews and presentations.

---

## Step 1 — Register (30 seconds)

- Go to https://placement-platform-nine.vercel.app
- Click **Create account**
- Enter name, email, password
- You're in — dashboard loads instantly with empty state
- Point out: stat cards, sidebar navigation, clean dark UI

---

## Step 2 — Upload Resume (60 seconds)

- Click **Resume** in the sidebar
- Upload your PDF resume
- Watch the processing happen in real time
- ATS Score appears — 0 to 100
- Detected skills shown as tags
- Point out the AI suggestions section:
  - Summary feedback
  - Missing skills
  - Specific improvements per section
  - Action items
  - Strengths identified

**Talking point:** "The ATS scorer uses a weighted formula across
keyword matching, contact info, word count, and section headers.
The AI suggestions are generated by Groq's Llama 3.1 via LangChain."

---

## Step 3 — Generate Career Roadmap (60 seconds)

- Click **Career Roadmap** in the sidebar
- Enter: "SDE-1 at a product startup"
- Click **Generate Roadmap**
- 8-week roadmap appears with topics per week
- Each week has specific topics, resources, and goals

**Talking point:** "The roadmap is dynamically generated based on
the target role. If a resume is uploaded, it personalises the plan
based on detected skill gaps."

---

## Step 4 — Track Applications (45 seconds)

- Click **Applications** in the sidebar
- Click **+ Add Application**
- Add 3 companies with different statuses:
  - Google — Interview Round 1
  - Amazon — Online Assessment
  - Microsoft — Applied
- Show the stats cards updating: Total, This Week, Offers, Offer Rate
- Show search and filter working

**Talking point:** "Status updates are instant with optimistic UI.
Stats are cached in Redis with 5-minute TTL and invalidated on writes."

---

## Step 5 — Dashboard Analytics (30 seconds)

- Click **Dashboard** in the sidebar
- Show stat cards with real data
- Show bar chart — applications by status
- Show line chart — weekly timeline with spike
- Hover sidebar to expand with labels

**Talking point:** "All chart data comes from a single aggregated
API endpoint. The sidebar hover-expand uses CSS transitions with
no JavaScript animation library."

---

## Step 6 — Settings (15 seconds)

- Click **Settings** gear icon
- Show Profile tab — name update
- Show Password tab — change password flow
- Show Notifications tab — toggle switches
- Show Danger Zone — DELETE confirmation

---

## Key Talking Points for Q&A

### Why Groq instead of OpenAI?
Free tier with generous limits. Llama 3.1 8B is sufficient for
resume analysis and roadmap generation. No credit card needed
during development.

### How does the RAG pipeline work?
PDF → PyMuPDF text extraction → LangChain text splitting →
fastembed embeddings → FAISS vector store → semantic search →
Llama 3.1 grounded answer.

### How does JWT auth work?
15-minute access tokens + 7-day refresh tokens. Refresh token
blacklisted on logout via SimpleJWT token blacklist table.
Rate limited to 5 login attempts per minute per IP.

### What is your test coverage?
38 tests across 5 apps, 82% coverage. GitHub Actions CI runs
on every push — Black formatting, Flake8 linting, pytest with
coverage gate at 70%.

### How would you scale this?
Read replicas for DB, horizontal Celery workers, Pinecone for
managed vector search, CDN for static assets, load balancer
in front of multiple Django instances.
